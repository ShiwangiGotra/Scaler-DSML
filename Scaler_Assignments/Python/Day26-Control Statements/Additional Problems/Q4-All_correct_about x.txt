Q4. All correct about x

You have been given the following piece of code. Assume that x has already been declared.

if x > 2:
    x = x*2
if x > 4:
    x = 0
print(x)

Select the correct statement regarding this code.

i) Output will always be equal to 0

ii) If x > 2 and x < 4, final value of x will be double the initial value

iii) For x in 0<x<4 the output will always be 0

iv) For x > 2, output will be equal to 0

Solution: iv
